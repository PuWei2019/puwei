# ec-site-springboot（開発途中）

本リポジトリは、個人で開発している EC サイトWeb アプリケーションの成果物です。  
主にバックエンドを中心に、フロントエンド（AJAX）およびテストまで含めて実装しております。

※ 本プロジェクトは現在も開発途中のため、未完成の機能が含まれております。

---

## 概要

本アプリケーションは、Spring Boot を用いた Web アプリケーションです。  
設計から実装、テストまでを一人で行い、  
バックエンドとフロントエンドの連携を意識して開発しております。

---

## 使用技術

- Java 17
- Spring Boot 3.4.12
- Maven 3.9.12
- Spring MVC
- AJAX（フロントエンドとの非同期通信）
- JUnit（テスト）
